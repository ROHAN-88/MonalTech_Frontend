# Monal Website

This repository contains the code for the Monal website, showcasing a fully responsive design with user-friendly navigation, dark mode, and optimized meta tags.

### Prerequisites

- Node.js (version 14 or higher)
- npm (Node Package Manager) or yarn
- Vite

## Features

- [x] Fully Responsive Design
- [x] User Friendly Navigation
- [x] Dark Mode
- [x] Metatags

## How to install

1. Clone this repositoy:

```bash
git clone git@github.com:monaltech/monal-website.git
```

2. Go into project

```bash
cd monal-landing-page
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
